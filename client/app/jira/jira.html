The following commands assume the slash command of '/jira'. To change this, see the Setup page.
  • /jira help - Displays the list of available commands.
  • /jira project info issue id - Displays information on the specified JIRA issue.
  • /jira project for jira username - Displays all in-progress tasks assigned to username.
  • /jira project top N [status] - Displays the top N priority items, optionally filtered by status.
  • /jira project todo - Displays all tasks that have not been started.
  • /jira project todo qa - Displays open tasks in "Ready for QA" status.
  • /jira project todo codereview - Displays tasks that have not been code reviewed.
  • /jira project todo po - Displays all tasks ready for PO acceptance.
