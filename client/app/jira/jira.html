The following commands assume the slash command of '/jira'. To change this, see the Setup page.
  • /jira help - Displays the list of available commands.
  • /jira info issue id - Displays information on the specified JIRA issue.
  • /jira for jira username - Displays all in-progress tasks assigned to username.
  • /jira top N [status] - Displays the top N priority items, optionally filtered by status.
  • /jira todo - Displays all tasks that have not been started.
  • /jira todo qa - Displays all tasks ready for qa assignment.
  • /jira todo codereview - Displays all tasks ready for code review.
  • /jira todo po - Displays all tasks ready for PO acceptance.
