The following commands assume the slash command of '/deployment'. To change this, see the Setup page.
  • /deployment add [task] - Adds a task for your team.
  • /deployment complete [task id] - Marks the specified task as complete.
  • /deployment me [task id] - Assigns the specific task to you.
  • /deployment assign [task id] [user name] - Assigns the specific task to the specified user. A blank user name is the same as the 'me' command.
  • /deployment delete [task id] - Deletes the specific task for your team.
  • /deployment info [task id] - Displays information about a specific task.
  • /deployment - Displays a list of deployment tasks for your team.
  • /deployment list - Displays a list of deployment tasks for your team.
  • /deployment done - Displays all completed tasks for your team.
  • /deployment doing - Displays a list of deployment tasks assigned to you.
  • /deployment assigned - Displays a list of in progress tasks.
  • /deployment unassigned - Displays a list of available tasks.
