The following commands assume the slash command of '/tasker'. To change this, see the Setup page.
  • /tasker help - Displays this list for you within slack.
  • /tasker add [task] - Adds a task for your team.
  • /tasker edit [task id] [task] - Edits the task name.
  • /tasker complete [task id] - Marks the specified task as complete.
  • /tasker me [task id] - Assigns the specific task to you.
  • /tasker assign [task id] [user name] - Assigns the specific task to the specified user. A blank user name is the same as the 'me' command.
  • /tasker delete [task id] - Deletes the specific task for your team.
  • /tasker info [task id] - Displays information about a specific task.
  • /tasker - Displays a list of tasks for your team.
  • /tasker list - Displays a list of tasks for your team.
  • /tasker done - Displays all completed tasks for your team.
  • /tasker doing - Displays a list of tasks assigned to you.
  • /tasker assigned - Displays a list of in progress tasks.
  • /tasker unassigned - Displays a list of available tasks.
NOTE: All display commands (/tasker info and below) can be broadcasted to the default channel by using 'shout'. For example, to display the task list to everyone in the channel, type /tasker shout list.
NOTE: All display commands (/tasker info and below) will be color-coded based on task status. Unassigned tasks will be gray, in progress (assigned) tasks will be orange, and completed tasks will be green.
