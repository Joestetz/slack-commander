The following commands assume the slash command of '/timer'. To change this, see the Setup page.
  • /timer help - Displays the list of available commands.
  • /timer [shout] start time description - Starts a timer, optionally displayed for entire channel with 'shout' option. Time must be in the format of h:m:s (ex: 2:10:0 or 0:0:30) and not exceed 12 hours. Anything longer than 12 hours should instead use slackbotReminders.
  • /timer edit timer id time - Edits the specified timer.
  • /timer end timer id - Ends the specified timer.
